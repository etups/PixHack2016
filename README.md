This project is for **2016 PIXNET HACKATHON Cloze Contest**.

## Start crawler
Use [scrapy](https://github.com/scrapy/scrapy) to download pixnet blog's articles.

```sh
./pixhack.sh --crawler
```

## Start jeiba

```sh
./pixhack.sh --jeiba
```

## Start test

```sh
./pixhack.sh --test
```

## Show database counts

```sh
./pixhack.sh --count
```
